# UK-Regional-Infrastructure-Inequality-Analysis

 Focused data cleaning and quality assurance project
 demonstrating handling of real world messy government
  datasets including mixed date formats, currency text
  conversion, broken referential integrity, and
  multi-table NULL recovery strategies.


## 💡 Key Skills Demonstrated

- Extensive Data Cleaning
- Data validation and integrity checking in SQL Server
- Aggregation, grouping, CTE and window functions in SQL
- Financial data analysis (budget,actual,variance)
- Mathematical calculations to fill in NULLs
